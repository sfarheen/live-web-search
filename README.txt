Live-web-search is a full stack web application. 

Technology:
Front-end: Jquery, Javascript
Back-end: C#
Database: SQL Server
IDE: Visual Studio .Net

The application architecture is:

LiveSearchServices - App server (Web Api)
LiveSearchUI - App client
SpeaerManager - App Business Layer 
Models - App data

This design of application leverages reusability.
With this seggregation of code and data, it is easy to add features, modify data etc.
